Sine Wave Audio Generator

An application for creating single-pitch sine wave audio files in the .wav format. Created using C# and WPF. (An example frequency to try out is 261.63 Hz, which in music is middle C.)

Download: SineWaveAudioGenerator_StevenHarolds.exe

Image Outline Maker

A GUI application that creates an outline-style image from an input image. There are various settings for color tolerances and distance ranges. Written in Java using JavaFX.

Download: ImageOutlineMaker_StevenHarolds.zip
(Note: This download includes all the necessary Java dependencies, so that they don't need to be installed separately. Run the application with ImageOutlineMaker.bat.)
