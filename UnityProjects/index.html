Steam Builder

This is the project I've spent the most time on so far. It is a 3D steampunk physics-based mechanical building game. Objects can be attached to each other to create variety of things, including buildings and moving vehicles.

Here are just a few details of how some of the game elements work: Axles are connected to other objects via physics joints. Gears have a capsule collider in each tooth which is optomized for operating with other gears. Engines provide torque to attached axles. Objects are attached to each other with a per-object grid system, meaning they are attached to the grid of the connected object instead of the world grid. This allows systems of connected objects to move freely in world space as long as they are not connected to the ground.

The game has two modes: a creative mode with an unlimited number of building components to build whatever you want, and a puzzle mode in which you are given a limited number of components to build a solution to a specific task.

Audio Animation

Another code-based audio visualization that I created. The music used is "Red Line" by Anna Yvette. Some objects, like the sun rays and the mountains, move according to the music volume, while the ribbons in the sky move according to which mid-range frequency band is currently the loudest.

Cube World Tanks

This is a game for mobile devices. Tanks move around a cube shaped world, and the player attempts to use their tanks to destroy the enemy tanks. It is a combination puzzle/battle game. Currently there is only one level and more details are needed, but the concept seems to work quite well.

Drag your finger across the playing field to rotate around the world. Buttons are displayed on the screen for the other controls.
